CSS 기본속성 복습2

margin, padding

margin은 border중심 외부 여백 padding은 내부여백
value값을 주는 방식은 top right bottom letf 순으로
margin : 10px 20px 10px 25px; 로 주던지
따로따로 margin-top:10px; 이렇게
주던가 상하 좌우가 같은 값이라면 margin: 10px 20px;로 주어도 된다.

EX) margin : 0px 20px 15px 25px;

EX) padding : 10px;

차이점은 margin은 컨텐츠의 요소 (예를 들면 백그라운드 컬러라든지)
를 포함하지 않고 padding은 포함한다

box-sizing, border

box-sizing은 테두리를 border중심 안쪽으로 생성할건지 바깥으로 생성할건지
기본 설정(default)는 box-sizing:content-box이며 테두리 바깥으로 생성
box-sizing:border-box는 안쪽으로 생성되므로 전체적인 크기가 증가하지 않아
더 유용하게 쓰임

border속성은 width,style,color등이 잇지만 그냥 묶어서
border: 5px solid red; 이렇게 써도됨.

box-sizing : border-box;
border : blue 5px dashed;

백그라운드는 background-image,size,repeat,attachment등이 있다.
