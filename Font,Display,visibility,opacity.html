CSS 기본속성 복습1

font-size 의 기본적 단위들은 px,em,pt가 있고

  1. px는 절대적으로 속성의 크기를 결정하고 font-size:12px;
  2. em은 상대적으로 속성의 크기를 결정하고 font-size:1.5em;
  3. pt는 절대적으로 속성의 크기를 결정한다.font-size:16pt;

display의 속성값은 기본적으로 none,block,inline,inline-block이 있다.

  1. none은 브라우저 화면에서 HTML요소를 제거
  2. block은 html요소를 block으로 출력
  3. inline은 마찬가지로 요소를 inline으로 출력
  4. inline-block은 inline속성과 동일하면서 높이와 폭값을 가질수있음.

visibility의 기본속성은 hidden과 visible(default값)이 있고 말그대로 숨기는속성

  1. visible : html요소를 출력
  2. hiddn : html요소를 출력하지않음

opacity는 투명도를 나타내며 0~1.0까지의 value를 가짐.

opacity는 0.7 따라서 투명도는 70%입니다.
